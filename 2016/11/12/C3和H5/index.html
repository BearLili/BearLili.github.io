HTML5

有语义化的标签

<nav></nav> 表示导航
<header> 表示页眉
<footer> 表示页脚
<section> 表示区块
<article> 表示文章 如文章、评论、帖子、博客
<aside> 表示侧边栏 如文章的侧栏
<figure> 表示媒介内容分组 与 ul > li 做个比较

注意:尽量避免全局使用header、footer、aside等语义标签

表单基本标签

  • input type=”email” 输入email格式
  • input type=”tel”手机号码
  • input type=”url” 只能输入url格式
  • input type=”number” 只能输入数字
  • input type=”search” 搜索框
  • input type=”range” 范围
  • input type=”color” 拾色器

下拉选项datalist标签

datalist要与input配合使用 datalist要有一个id名 然后在input中用一个list属性链接这个id

度量器meter标签

meter用作度量器 不建议用于进度条

在有min和max的情况下，
value< low 显示绿色
value> high 显示红色
low≤value≤high 显示红色
在没有min和max的情况下，
都显示绿色

加密keygen标签

生成加密字符串
公钥和私钥

表单属性

  • placeholder 占位符
  • autofocus 获取焦点
  • multiple 文件上传多选或多个邮箱地址
  • autocomplete 自动完成，用于表单元素，也可用于表单自身
  • form 指定表单项属于哪个<form>，处理复杂表单时会需要
  • novalidate 关闭验证，可用于<form>标签
  • required 验证条件，必填项
  • pattern 正则表达式 验证表单

表单事件

  • oninput 用户输入内容时触发，可用于移动端输入字数统计
  • oninvalid 验证不通过时触发

音频audio标签

autoplay 自动播放
controls 是否显不默认播放控件
loop 循环播放
preload 预加载 同时设置autoplay属性时 preload失效

视频video标签

视频的属性

  • autoplay 自动播放
  • controls 是否显示默认播放控件
  • loop 循环播放
  • preload 预加载，同时设置了autoplay，此属性将失效
  • width 设置播放窗口宽度
  • height 设置播放窗口的高度
  • currentTime 视频播放的当前进度、
  • duration:视频的总时间
  • paused:视频播放的状态.

视频的方法

  • load() 加载
  • play() 播放
  • pause() 暂停

视频的事件

  • oncanplay: 事件在用户可以开始播放视频/音频（audio/video）时触发。
  • ontimeupdate:通过该事件来报告当前的播放进度.
  • onended:播放完时触发

获取标签的方式

  1. document.getElementsByClassName
    通过类名的形式获取元素，返回一个伪数组
  2. document.querySelector
    通过获取CSS选择器获取元素，比如.class #id,返回的是第一个DOM
  3. document.querySelectorAll
    通过获取标签来获取元素，返回的是一个伪数组

操作类名的属性

  • Dom.className.add 添加类名
  • Dom.className.remove 移除类名
  • Dom.className.toggle 切换类名
  • Dom.className.contains 是否含有类名

通过data-自定义名字=”内容”来设置
然后通过Dom.dataset [“自定义的名字”]

注意:如果设置的自定义的名字是a-b的形式，则用驼峰法获取，例如
通过data-init-set=””设置,则通过Dom.dataset [“initSet”]

拖拽属性

要拖拽的属性需要加上 draggable=”true” 属性

拖拽元素

  • ondrag 应用于拖拽元素，整个拖拽过程都会调用
  • ondragstart 应用于拖拽元素，当拖拽开始时调用
  • ondragleave 应用于拖拽元素，当鼠标离开拖拽元素时调用
  • ondragend 应用于拖拽元素，当拖拽结束时调用

目标元素

  • ondragenter 应用于目标元素，当拖拽元素进入时调用
  • ondragover 应用于目标元素，当停留在目标元素上时调用e.preventDefalut(): 阻止默认行为的执行
  • ondrop 应用于目标元素，当在目标元素上松开鼠标时调用
  • ondragleave 应用于目标元素，当鼠标离开目标元素时调用

web存储

特性

(1)设置、读取方便

(2)容量较大

(3)只能存储字符串，可以用Json.stringify()转换

window.sessionStorage

(1)生命周期为关闭浏览器窗口

(2)在同一个窗口下数据可以共享

window.localStorage

(1)永久生效，除非手动删除

(2)可以多窗口共享

方法

setItem(key, value) 设置存储内容

getItem(key) 读取存储内容

removeItem(key) 删除键值为key的存储内容

clear() 清空所有存储内容

CSS3

属性选择器

  1. div[class] 表示存在attr属性即可；

  2. div[class=mydemo]表示属性值完全等于val；

  3. div[class*=mydemo]表示的属性值里包含val字符并且在“任意”位置；

  4. div[class^=mydemo] 表示的属性值里包含val字符并且在“开始”位置；

  5. div[class$=demos] 表示的属性值里包含val字符并且在“结束”位置；

伪类选择器

这里面的选择器都是相对于它的父元素找到对应的div

  1. div:first-child 第一个子元素
  2. div:last-child 最后一个子元素
  3. div:nth-child(n) n是指第几个元素 n不是索引值 n≥0;
  4. div:nth-last-child(n) 倒着寻找 与nth-child相似
  5. div:nth-child(odd) 所有的奇数 odd三个字母 所以是奇数
  6. div:nth-child(even) 所有的偶数 even四个字母 所以是偶数
  7. div:empty 选中没有任何子节点的元素，包括空格

透明度

  • opactiy
    opactiy可以设置透明度，但是如果给父盒子设置了opactiy，那么字盒子也会继承透明度。
  • transparent
    transparent可以设置透明度，但是固定的不能修改。
  • rgba和hsla
    rgba(r,g,b,a) r:红色,g:绿色,b:蓝色,a:透明度。r,g,b的取值范围0~255,a取值0~1。
    hsla(h,s,l,a) h:色调,s:饱和度,l:亮度,a:透明度。
    h的取值范围0~360,依次按照红绿蓝的顺序各占三分之一。
    s的取值范围0~100%,
    l的取值范围0~100%,
    a取值0~1。

文本阴影text-shadow

text-shadow(水平位移，垂直位移，模糊度，颜色)

  • 水平位移:正值向右，负值向左。
  • 垂直位移:正值向下，负值向上。
  • 模糊度 :模糊度不能为负值，值越大越模糊。
  • 颜色 :可以设置透明度。

盒模型box-sizing

  • border-box
    设置的border、padding都不计入盒子的大小。
  • content-box
    设置的border、padding都计入盒子的大小。
  • padding-box
    设置的padding计入盒子的大小。

边框border

  • border-raduis边框圆角
    左上，右上，右下，左下的顺序依次设置
  • box-shadow边框阴影
    水平位移，垂直位移，模糊程度，阴影大小，颜色，内外阴影
    内外阴影默认为外阴影，即outset,不用设置;如果要设置内阴影，则使用inset。
  • border-image边框图片
    (1).border-image-source:url();
    引入图片。
    (2).border-image-width:
    图片的宽度，要带单位PX。
    (3).border-image-slice:
    裁剪，不用带单位PX。
    (4).border-image-repeat:图片平铺
    border-image-stretch:图片拉伸
    border-image-round:图片均匀平铺，图片先计算再平铺。

背景图片background

  • background-size图片尺寸
    (1).可以写具体的数值大小
    (2).可以写百分比
    (3).cover 覆盖 图片会铺满盒子，但有可能不按比例。
    (4).contain 包含 图片会按比例，但有可能铺不满盒子。
  • background-origin背景设置起点
    (1).border-box 图片会从边框开始设置。
    (2).padding-box 图片会从去除边框的位置开始设置。
    (3).content-box 图片会从去除边框以及内边距的位置开始设置。
  • background-clip背景设置起点
    (1).border-box 图片会从边框开始设置。
    (2).padding-box 图片会从去除边框的位置开始设置。
    (3).content-box 图片会从去除边框以及内边距的位置开始设置。

线性渐变linear-gradient

径向渐变radial-gradient

过渡transition

  • transition:过渡属性、持续时间、运动曲线、延迟执行时间;

注意:transition要添加给要过渡的标签的本身,过渡属性和持续时间是必须的不能省略

  • transition-property:过渡的属性。
  • transition-duration:过渡的事件。

  • transition-timing-function：时间函数

    linear:线性匀速。
    ease-in:加速。
    ease-out:减速。
    ease-in-out:先加速后减速。

    • transition-delay:过渡延迟执行。

变换transform

transform要与transition配合使用。

  1. 缩放scale
    scale() 传一个值得时候，是等比例缩放，大于1是放大，小于1是缩小。
    传两个值得时候，x是水平缩放，y是垂直缩放。
  2. 位移translate
    (1).一个参数 单位px，默认水平方向。
    (2).两个参数 单位px，水平和竖直方向。
    (3).百分数
    小技巧:让定位的盒子居中的方法
    **left:50%**
**transform:translate(-50%)**
  3. 旋转rotate
    rotate(角度)
    单位:deg
    正值:顺时针
    负值:逆时针
  4. 旋转中心:origin
    默认中心为center，center。

3D

  1. 透视效果
    prespective 透视
    设置给父盒子，子盒子会有3D效果
    但不常用。
  2. 盒子背面隐藏
    backface-visility:hidden
  3. 3D效果
    transform-style:preserve-3d。
    设置给父盒子，子盒子会有3D效果
    transform:translate(z)
    要配合transform-style:preserve-3d一起使用才有效果。

伸缩布局flex

  • 必要元素：

    (1).指定一个盒子为伸缩盒子 display: flex

    (2).设置属性来调整此盒的子元素的布局方式 例如 flex-direction

    (3).明确主侧轴及方向

    (4).可互换主侧轴，也可改变方向

  • flex-direction 调整主轴方向(默认为水平方向)

    (1).row 水平方向

    (2).reverse-row 反转

    (3).column 垂直方向

    (4).reverse-column 反转列

  • justify-content 对齐方式

    (1).flex-start 起点对齐

    (2).flex-end 终点对齐

    (3).center 中心对齐

    (4).space-around 均匀平分对齐

    (5).space-between 先两端对齐，然后均匀平分

  • flex 控制子项目的缩放比例
    设置flex的子项目按所设置的数值分配比例

  • order 控制子项目的排列顺序
